Luminism Shadow Box

About this project

The Liuminism Shadow Box features hand-drawn artwork in a frame that houses a light sensor and LEDs. In the light, it appears to be just a framed picture. However, when the lights go out, the light sensor activates the LEDs to reveal secret elements behind the artwork.

What you'll learn

  • Part creation using relations and dimensioning
  • Advanced assemblies using efficient mates
  • Large scale exploded views and bill-of-materials
  • Defining and organizing complex sketches
