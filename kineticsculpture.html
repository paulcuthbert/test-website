Kinetic Sculpture

About this project

The Kinetic Sculpture features two discs that create a unique visual design, called a moire pattern, while rotating. These moire patterns are backlit by an RGB LED strip to further accentuate their affect.

What you'll learn

  • Part creation using relations and dimensioning
  • Advanced assemblies using efficient mates
  • Large scale exploded views and bill-of-materials
  • Defining and organizing complex sketches
